# Feature delta benchmark

- **Scene:** Cornell box
- **Resolution:** 640x360
- **Samples (baseline):** 4
- **Timed reps (median):** 5, warmup: 2
- **OpenMP threads:** 2

| Configuration | Median time (s) | vs baseline scalar | MRays/s (est.) | Notes |
|---|---:|---:|---:|---|

| Reflections OFF | 0.3463 | -29.8% time | 2.66 | no recursive bounce path |

## How to read **vs baseline**

- **Scalar rows:** *% time* = `100 * (t - t_baseline) / t_baseline` — **negative** means faster than baseline.
- **SIMD / wavefront / Morton:** *% faster* = `100 * (t_baseline / t - 1)` — **positive** means faster than baseline scalar.
- **BVH / frustum / stratified:** same scalar *% time* semantics as other scalar toggles.
- **Adaptive:** uses half the primary samples; MRays/s uses the effective sample count.
- **Scaling tables** (`--sweep`): *% time* vs the reference row named in each subsection (resolution vs 640w, SPP vs 4, **max_depth vs 5**, threads vs first row).
