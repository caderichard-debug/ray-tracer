Build & run

Minimal steps copied from the repository README. See README.md for the full control reference and platform notes.

Prerequisites

macOS (Homebrew)

brew install gcc sdl2 sdl2_ttf

Ubuntu / Debian

sudo apt update
sudo apt install build-essential libsdl2-dev libsdl2-ttf-dev libomp-dev

Windows (MSYS2)

pacman -S mingw-w64-x86_64-gcc mingw-w64-x86_64-SDL2 mingw-w64-x86_64-SDL2_ttf

Build & run targets

# CPU batch (feature flags via Makefile)
make batch-cpu
make run

# CPU interactive (recommended first run)
make interactive-cpu
make runi-cpu

# ASCII terminal
make ascii
make runa

make interactive and make runi are aliases for the CPU interactive targets.

Regression check

Deterministic batch output is hashed against golden PPMS in tests/regression/:

make regression-test

First launch tip

When the interactive binary starts, press H for the on-screen help overlay covering movement, quality keys, analysis modes, and screenshot shortcuts.
