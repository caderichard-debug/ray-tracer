CPU-first SIMD ray tracer

A from-scratch C++ path built around AVX2 and OpenMP, with a real-time SDL2 interactive mode, high-quality batch output, and a retro ASCII renderer.

Build & run Source on GitHub
GPU mode is work in progress. This project is documented and optimized around the CPU renderer; treat the OpenGL path as experimental until parity and benchmarks are stable.

Cornell box (CPU batch)

Phong shading, hard shadows, and reflections—same scene family as the interactive SDL view, rendered offline for comparison.

Why this exists

The codebase is a learning and performance lab: SIMD-friendly shading, optional progressive and adaptive sampling, wavefront tiling, post denoise controls in interactive mode, and a deterministic regression harness for batch output.
